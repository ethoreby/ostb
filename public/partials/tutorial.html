Getting started with Markdown

Markdown is a simple text formatting system that makes it easy to write content for the web. However, it works a bit differently from what you might be used to, so we've created this short tutorial/reference to get you going.

Headings

To create a heading, insert a hash symbol (#) in front of the text you want to use as the heading. A single hash will create a large heading, additional hashes will decrease it in size, to a minimum of six.

Paragraphs

To create a paragraph, seperate text from prior text with two returns. If you would like to start content on a new line but without the extra space, put two spaces at the end of your current line.

Images

To insert images, use the following format:

![Alt text](image address)

Lists

Lists are created with dashes, stars or pluses at the beginning of each line. To create sublists, indent a line with a tab (or several).
