Create open source textbooks to share with the world.

Write them collaboratively – with anyone.

Host them anywhere – no platform lock-in.

Why limit yourself to paper, physical textbooks? A textbook as a website can be accessed on any number of devices, anywhere in the world, by any student with the desire to learn.

Coming generations will learn primarily from such resources – do you want to be one of the few that create them?

No more emailing word documents! textbooks.io projects can be written in the browser, collaboratively, even in real-time – just invite your colleagues to the project you want to work on them with.

Projects can also be contributed to if you’re not a listed contributor – just open one up, start editing, and the changes you make will be sent to the project owner when you’re done (who can choose to accept them).

textbooks.io is not a platform for hosting books, only for writing them. Instead, a download link is provided for anything you create, allowing users to deploy their book anywhere they want.

So if textbooks.io shuts down one day, you can be sure that the book you created here will continue to work just fine.
